Welcome to

My Incredible, Wonderful, So Good, Very Awesome Webpage

By Steven Chu

What's going on? (Marvin Gaye, duh)

Hi, folks! I created this website as part of an assignment for my Web Analytics class. The goal of this class is to understand how the Internet works. Here's what I've learned so far: you click on some stuff, some other stuff happens, and everyone who was involved in the making of allllllllllll that stuff knows everything that just happened. Fun, right?

Anyway, here's how you can help me pass this class: click on everything on this website. If you do that, I will be able to see what you clicked on, when you clicked on it, what type of device you were using when you clicked on it, and all the other information you could possibly not want me to know. Creepy, I know, but help me out.

If you have any suggestions on how I can make this website better, please let me know. If you are a real person reading this (as opposed to a computer-programmed webcrawler like all of Russia), then you probably have my personal email address. Holler at me that way. Or, if you want to be all professional and stuff, then reach out via the contact page, though I'm not entirely sure the submit button actually does anything... Oh well, try anyway if you want!
